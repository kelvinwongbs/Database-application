# Implement a Ridesharing System using MySQL and Java

### Course: Introduction to Database Systems (Group Project)

**Functionalities:**
1. Administrator
   * Create tables
   * Delete tables
   * Load data
   * Check data
2. Passenger
   * Request a ride
   * Check trip records
3. Driver
   * Search requests
   * Take a request
   * Finish a trip
4. Manager
   * Find (finished) trips
